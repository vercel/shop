# Vercel Shop Monorepo

Monorepo for the [Vercel Shop](https://vercel.shop) storefront template, documentation site, and supporting tooling.

| App | Description |
|-----|-------------|
| `apps/template` | The Next.js storefront template — see its [README](apps/template/README.md) |
| `apps/docs` | Documentation site at [vercel.shop](https://vercel.shop) |
| `apps/cli` | `create-vercel-shop` scaffolding CLI |

Built with [Turborepo](https://turbo.build/) and pnpm.
