The changelog records template changes that affect downstream storefronts — user-visible behavior, routes, config or environment variables, caching and cache invalidation, data contracts, runtime-affecting dependency bumps, and breaking changes. Entries aggregate into weekly releases below.

Downstream storefronts can upgrade entry-by-entry with the [`update-shop` skill](/docs/skills/update-shop), which reads these entries directly.
