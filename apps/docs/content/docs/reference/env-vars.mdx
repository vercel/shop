---
title: Environment Variables
description: Required and optional environment variables for Vercel Shop.
type: reference
---

## Required

| Variable | Description |
|----------|-------------|
| `COMMERCE_STORE_DOMAIN` | Your store domain, e.g. `your-store.example.com`. Found in your provider's admin settings. |
| `COMMERCE_API_TOKEN` | Public API access token for storefront requests. Found in your provider's admin under API credentials. |
| `NEXT_PUBLIC_SITE_NAME` | The name displayed in your storefront's header and metadata. |

## Optional

| Variable | Description |
|----------|-------------|
| `COMMERCE_API_PRIVATE_TOKEN` | Private API token for server-side requests. Enables higher rate limits and access to draft content. |
| `COMMERCE_CUSTOMER_ACCOUNT_URL` | Customer Account API URL. Required when using the enable-auth skill. |
| `COMMERCE_CUSTOMER_CLIENT_ID` | Customer Account API client ID. Required when using the enable-auth skill. |
| `COMMERCE_CUSTOMER_CLIENT_SECRET` | Customer Account API client secret. Required when using the enable-auth skill. |
