## Pages

| Route                   | File                                | Description                                                                                        |
| ----------------------- | ----------------------------------- | -------------------------------------------------------------------------------------------------- |
| `/`                     | `app/page.tsx`                      | Home page with hero, products grid, info section                                                   |
| `/products/[handle]`    | `app/products/[handle]/page.tsx`    | Product detail page with Shopify's default selection                                               |
| `/collections/[handle]` | `app/collections/[handle]/page.tsx` | Collection listing with filters, sort, pagination                                                  |
| `/collections/all`      | `app/collections/[handle]/page.tsx` | Virtual "All Products" listing; synthesized since Shopify's Storefront API has no `all` collection |
| `/search`               | `app/search/page.tsx`               | Search results (same grid as collections)                                                          |
| `/cart`                 | `app/cart/page.tsx`                 | Full cart page with upsells                                                                        |
| `/pages/[slug]`         | `app/pages/[slug]/page.tsx`         | CMS-driven marketing pages                                                                         |
| `/about`                | `app/about/page.tsx`                | About page                                                                                         |

## Variant URLs

PDP option and cart links use Shopify's numeric `variant` query parameter. This keeps the public selection space finite while Shopify still resolves high-variant products and Combined Listings without loading every variant.

| URL                                           | Purpose                                                              |
| --------------------------------------------- | -------------------------------------------------------------------- |
| `/products/:handle`                           | Base PDP with Shopify's default selection                            |
| `/products/:handle?variant=:variantId`        | Concrete variant URL; selection regions resolve within the PPR route |
| `/products/:combinedChild?variant=:variantId` | Combined Listing variant using the owning child product handle       |

Arbitrary option-name query parameters do not select a variant. Progressive partial selection is an opt-in storefront capability rather than a default route behavior.
