The template connects to Shopify through the Storefront API. This page is an orientation map from Shopify concepts to the template routes and docs that cover them — start here, then follow the links for the mechanics.

For token setup and required scopes, see [Getting Started](/docs/getting-started) and [Storefront API Permissions](/docs/reference/storefront-api-permissions). For the API client and query patterns, see [Storefront API](/docs/reference/storefront-api).

## Concept mapping

| Shopify concept | Template route          | What renders it                                                    | Docs                                                     |
| ---------------- | ------------------------ | -------------------------------------------------------------------- | --------------------------------------------------------- |
| Products         | `/products/[handle]`    | Product detail page with variants, gallery, buy section             | [PDP anatomy](/docs/anatomy/pages/pdp), [Product setup](/docs/shopify/pdp) |
| Collections      | `/collections/[handle]` | Product grid with filtering, sorting, pagination                    | [PLP anatomy](/docs/anatomy/pages/plp)                    |
| Menus            | Navigation, footer       | Quick links bar (with hover panels), mobile sheet, footer columns   | [Navigation](/docs/anatomy/navigation), [Enable Shopify menus](/docs/skills/enable-shopify-menus) |
| Pages            | `/pages/[handle]`       | Localized Shopify Page rich text and SEO metadata                   | [Content pages](/docs/anatomy/pages/content)              |
| Policies         | `/policies/[handle]`    | Configured store policies, also linked from the footer              | [Footer](/docs/anatomy/footer)                            |
| Search           | `/search`                | Same grid as collections, driven by query param                     | [PLP & Search anatomy](/docs/anatomy/pages/plp)           |
| Cart             | `/cart` + overlay        | Cart context with optimistic updates                                 | [Cart](/docs/anatomy/cart)                                |

## Menus

By default, the template uses hardcoded navigation links and an empty footer. Use [`/vercel-shop:enable-shopify-menus`](/docs/skills/enable-shopify-menus) to fetch `main-menu` and `footer` menus from **Online Store → Navigation** in Shopify admin instead. See [Navigation](/docs/anatomy/navigation) for how the nav renders.

## Cache invalidation

Shopify data is cached (`"use cache"` or `"use cache: remote"`) rather than fetched fresh on every request. Configure Shopify webhooks pointed at `/api/webhooks/shopify` so edits in Shopify Admin invalidate the right cache tags instead of waiting for the cache lifetime to expire. See [Webhooks](/docs/anatomy/webhooks) for the endpoint, supported topics, and setup steps.
